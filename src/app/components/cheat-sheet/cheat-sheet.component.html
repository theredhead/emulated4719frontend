Cheat sheet Usage and opcodes

Use the code editor to type opcodes or load one of the example programs. Hover over buttons to see what they do.

Examples opcodes
1 byte instructions
halt
halts the processor
add
R0 = R0 + r1
sub
R0 = R0 - R1
inc0
R0 = R0 + 1
inc1
R1 = R1 + 1
dec0
R0 = R0 -1
dec1
R1 = R1 - 1
bell
Ring a bell
2 byte instructions
prn
Print the value in the next byte
ld0
Load the value of the next byte into R0
ll1
Load the value of the next byte into R1
st0
Store the value of R0 into the address stored in the next byte
st1
Store the value of R1 into the address stored in the next byte
jmp
Jump to the address pointed to by SS
jz
Jump to the address stored in the next byte
jnz
Jump to the address stored in the next byte
