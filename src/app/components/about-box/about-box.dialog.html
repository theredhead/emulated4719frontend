About this frontend

Some information about this sotware

The 4719 is a (fictional) processor that was introduced to me by a man I've never met; Richard Buckland. I have no idea about the history of the CPU.

Some years ago, probably 2017 or so, while browsing iTunes U, I came across a UNSW course that grabbed my interest. COMP1917. The UNSW had been so amizingly cool as to make videos of their lectures widely available. In some of the earlier lectures the fictional processor makes its' debut.

This web application simulates that cpu.

Thank you Richard Buckland and UNSW

This software is freely available.
